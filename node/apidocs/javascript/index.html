WeDeploy

src/api/WeDeploy.js

The main class for making api requests. Sending requests returns a promise that is resolved when the response arrives. Usage example:

WeDeploy
  .url('/data/tasks')
  .post({desc: 'Buy milk'})
  .then(function(response) {
    // Handle response here.
    console.log(response.body())
  });
new WeDeploy(url: string, paths: ...string)
Parameters
url (string) The base Url
paths (...string) Any amount of paths to be appended to the base Url
